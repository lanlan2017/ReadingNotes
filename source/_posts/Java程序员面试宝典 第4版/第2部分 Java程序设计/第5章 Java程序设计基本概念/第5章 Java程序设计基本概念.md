# 第5章 Java程序设计基本概念 #
对于一个求职者或者应届毕业生来说, 公司除了对项目经验有所问询之外, 最好的考试办法就是检查基本功, 包括编程风格, 以及对赋值语句、 递增语句、 类型转换、 数据交换等程序设计基本概念的理解。
